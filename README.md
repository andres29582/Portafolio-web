# Portfólio Web

Portfólio web profissional desenvolvido para apresentar meu perfil como estudante de Análise e Desenvolvimento de Sistemas, reunindo projetos, habilidades técnicas, formação, experiência profissional e minha forma de pensar sistemas.

O projeto foi construído com Next.js, TypeScript e Tailwind CSS, com uma estrutura modular, conteúdo separado dos componentes visuais e suporte inicial a múltiplos idiomas.

## Objetivo

Criar uma apresentação mais estruturada do que um currículo tradicional, mostrando não apenas informações pessoais, mas também organização técnica, evolução profissional e visão de arquitetura de sistemas.

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
